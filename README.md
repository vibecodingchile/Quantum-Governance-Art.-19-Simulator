# Quantum Governance & Art. 19 Simulator
Este repositorio contiene un laboratorio virtual para auditar el **vacío técnico** de la computación cuántica.

## Objetivo
Demostrar cómo la incertidumbre cuántica afecta la seguridad jurídica y las garantías constitucionales (Art. 19). 

## Uso
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar simulador: `python main.py`
